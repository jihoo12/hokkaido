hokkaido — LLVM-based compiler with cubical compile-time evaluation

## installation
```sh
git clone https://github.com/hokkaido-lang/hokkaido.git
mkdir build
cd build
cmake ..
make
```
###
**requirements**  
clang  
cargo  
llvm
